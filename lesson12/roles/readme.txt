В роли epel только один простой таск - установка epel

В роли myserver есть:
1. Зависимость от роли epel в meta/main.yml
2. Переменная sysctl_settings в vars/main.yml
3. Таск, который выполняет 2 задачи:
3.1. Устанавливает инструменты для мониторинга производительности с помощью модуля yum: htop, atop, net-tools, sysstat, iotop
3.2. Задаёт значения для некоторых sysctl-настроек
